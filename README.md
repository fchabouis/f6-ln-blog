# F6-LN - Blog de voyage à vélo

Le code de site https://f6-ln.chabouis.fr/

Blog de voyage à vélo de F6-LN, relatant un tour du monde à vélo d'un an (août 2008 - août 2009).

Ce blog était initialement hébergé sur [my.opera.com/f6-ln](http://my.opera.com/f6-ln/blog/), la plateforme de blogs de Opera. Suite à l'arrêt de ce service par Opera, le blog a été remis en ligne début 2026 grâce à une sauvegarde réalisée à l'époque.
